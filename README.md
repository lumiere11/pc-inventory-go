
# PC parts inventory in go

This is a backend for a pc parts inventory made with Go + Sqlite


## Prerequisites

- [Go](https://go.dev/dl/) 1.20+ installed  
- Git installed  

## Clone the Repository

```bash
git clone https://github.com/lumiere11/pc-inventory-go
cd pc-inventory-go
```
## Build the project
```bash
go build -o app .
#Run it
./app

```

## Tests
```
go test ./...
```
